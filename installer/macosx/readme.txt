Unipro UGENE on Mac OS X
========================

The Unipro UGENE disk image contains "Unipro UGENE" application
and links to the following directories:
  * Applications
  * Samples

To start UGENE from the package, double-click on the "Unipro UGENE" icon.
To install UGENE on your system drag the "Unipro UGENE" icon to the "Applications"
directory or another desired location.

The "Samples" directory contains UGENE sample files in different formats.
For example, sequences in FASTA, GenBank and other formats.
If you have installed UGENE on your system, you can also drag the "Samples" link
to the same location to be able to access the UGENE samples data.
